KHANAAS

Khan As A Service


v0.0.2

Introduction

KHANAAS provides a way to express general disdain when something doesn't go quite right. Maybe your co-worker broke the build or it's laundry day again.

API

/kirk/:name/

Will return content of the form ':nameeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeee' with William Shatner (as Captain Kirk) screaming in the background.

/spock/:name/

Will return content of the form ':nameeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeee' with Zachary Quinto (as Spock) screaming in the background.

/jones/:name/

Will return content of the form ':Name. I hate :Name.' with Harrison Ford (as Indiana Jones) bickering in the background.

KHANAAS is a fork of FOAAS (Fuck Off As A Service). See FOAAS on github.
